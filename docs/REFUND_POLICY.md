# GOAT Refund Policy

## Refund Terms for Per-Project Content Preparation Services

**Last Updated: January 6, 2026**

---

## 1. Introduction

GOAT operates on a **per-project, non-subscription model**. Each project is a separate service engagement with clear deliverables and pricing. This Refund Policy explains when and how refunds are provided for GOAT's content preparation services.

---

## 2. Refund Eligibility

### Automatic Refund Conditions

#### Technical Failures
- **Service Unavailability**: If GOAT systems are unavailable for more than 24 hours during your project timeline
- **Data Corruption**: If your uploaded content is corrupted during processing
- **Processing Errors**: If our AI systems produce unusable or severely defective output
- **Format Failures**: If deliverables do not meet the specified format requirements

#### User Error Refunds
- **Incorrect File Upload**: If you upload the wrong file and request refund within 1 hour
- **Duplicate Projects**: If you accidentally submit the same project twice
- **Account Issues**: If billing errors occur due to GOAT account problems

### Partial Refund Conditions

#### Quality Issues
- **Minor Defects**: 50% refund for deliverables with correctable quality issues
- **Incomplete Processing**: Pro-rated refund based on uncompleted portions
- **Format Inconsistencies**: 25% refund for minor formatting deviations

#### Timeline Delays
- **Minor Delays**: No refund for delays under 24 hours
- **Extended Delays**: 25% refund for delays of 24-72 hours
- **Major Delays**: 50% refund for delays over 72 hours

---

## 3. Non-Refundable Conditions

### Content-Related Issues
- **User Dissatisfaction**: Changes of mind or dissatisfaction with AI-enhanced content
- **Creative Differences**: Disagreement with AI processing choices or style decisions
- **Expectation Mismatches**: Content not meeting unstated expectations
- **Alternative Preferences**: Preference for different processing approaches

### User Responsibility
- **Incorrect Instructions**: Providing wrong processing instructions or requirements
- **Incomplete Information**: Failing to provide necessary context or source materials
- **File Format Issues**: Uploading unsupported file formats after confirmation
- **Account Problems**: Issues caused by user account security or access problems

### Technical Limitations
- **AI Limitations**: AI systems cannot perfectly replicate human creative decisions
- **Content Complexity**: Extremely complex content may have limitations in processing
- **Source Quality**: Poor quality source materials affecting final output
- **Technical Constraints**: Platform limitations for certain content types

---

## 4. Refund Process

### Request Submission
- **Time Limit**: Refund requests must be submitted within 7 days of project completion
- **Contact Method**: Submit requests via support@goatvault.io or the support portal
- **Required Information**: Include project ID, reason for refund, and supporting evidence
- **Response Time**: GOAT will acknowledge receipt within 24 hours

### Review Process
- **Initial Review**: Support team reviews request within 48 hours
- **Evidence Evaluation**: GOAT evaluates provided evidence and project records
- **Technical Assessment**: Technical team assesses any claimed processing issues
- **Decision Communication**: Refund decision communicated within 5 business days

### Refund Methods
- **Original Payment Method**: Refunds processed to the original payment method
- **Processing Time**: 5-10 business days for refund processing
- **Partial Refunds**: Calculated based on the percentage of unusable deliverables
- **Credit Options**: Users can request GOAT credit instead of cash refund

---

## 5. Refund Amounts

### Full Refunds (100%)
- **Service Failure**: Complete system failure preventing project completion
- **Data Loss**: Irrecoverable loss of user content due to GOAT error
- **Security Breach**: Unauthorized access to user content due to GOAT security failure
- **False Advertising**: Services not matching advertised capabilities

### Partial Refunds
- **Quality Issues**: 25-75% based on severity of defects
- **Timeline Issues**: 10-50% based on delay duration
- **Incomplete Delivery**: Pro-rated based on undelivered portions
- **Format Issues**: 25% for correctable formatting problems

### No Refunds
- **User Error**: Issues caused by user mistakes or incorrect usage
- **Expectation Issues**: Content not meeting subjective user expectations
- **Third-Party Issues**: Problems caused by third-party integrations
- **Account Issues**: Problems due to user account management
